Pour la bonne exécution du programme veillez à ce que la version 5.9 de Qt soit installée .
Pour cela il faut suivre les instructions sur cette page : https://linuxhint.com/install-qt-5-9-1-qt-creator-4-3-1-ubuntu/
Après l'installation, il faut modifier le Bashrc pour qu'il prennent en compte uniquement cette version et pas les autres.
Il faut aussi que les librairies suivantes: -lopengl32
LIBS ,-lglu32
LIBS ,-lglut32
